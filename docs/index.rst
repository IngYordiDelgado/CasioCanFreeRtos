Software Detail Design
======================


Welcome to Casiocan Documentation, the proyect is designed to to run on board Nucleo-G0B1 which has
built in the microcontroller stm32g0b1re. The proyect is pretty simple it is just an programmable clock
with a serial port interface and a LCD

To build proyect just type `make build` and then to flash the board `make flash` it is
